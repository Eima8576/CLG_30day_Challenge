Flex Box

Flex box is one dimensional layout method for arranging data in rows and columns. Flex box allows us to create awesome layout with it's amazing feature of arranging items.

Flex Container Properties

flex-direction

The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).

flex-wrap

The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines.

flex-flow

flex-flow property is the shorthand for flex-direction and flex-wrap properties

justify-content

The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.

align-items

It controls the alignment of items on the Cross Axis.

align-content

The CSS align-content property sets the distribution of space between and around content items along a flexbox's cross-axis.
